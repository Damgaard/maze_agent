# Claude Code Instructions

## Required Reading at Startup

Before working on this project, you must read the following documents in order:

1. **`docs/shared_context/01_understanding_shared_context.md`** - Shared cross-project documentation (environment, constraints, standards)

The shared context file is the entry point for all shared documentation across projects. It explains what additional documentation is available and when to read it based on your task.

2. **`docs/01_project_overview.md`** - Information specific to this project ( architecture, goals, constraints, cost control)
