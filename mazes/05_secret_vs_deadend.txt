=== MAZE 05: Secret vs Dead End ===
Door south leads to dead end. Secret door north leads to exit.
South room only has a door back north (the one you entered through).

#####
# E #
#   #
#####
  X    <- Secret Door North (hidden, found by searching)
#####
#   #
# S #
#####
  #    <- Door South
#####
#   #
#   #
#####

Legend:
S = Start position
E = Exit position
# = Wall
X = Secret door (not visible without searching)
(space) = Open floor

Note: Bottom room is a dead end - only has door north back to start.
